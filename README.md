# Кутищева Ксения
## О программе
Система для управления рецептами. Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.
Система позволяет:
- Создавать ингредиенты с названием, количеством и единицей измерения
- Создавать рецепты с набором ингредиентов
- Масштабировать рецепты на нужное количество порций
- Создавать диетические рецепты (веган, без глютена и т.д.)
- Формировать список покупок из нескольких рецептов
- Объединять списки покупок

## Установка и запуск
1) Откройте терминал. 
2) Для начала склонируйте репозиторий на компьютер: введите ``` bash git clone https://github.com/kutishchevavkseniya-png/Hw2_TP ```, затем нажмите Enter. Дождитесь завершения клонирования. 
3) Перейдите в папку проекта: введите ``` bash cd Hw2_TP``` и нажмите Enter. 
4) Установите зависимости: введите ``` pip install -r requirements.txt``` и нажмите Enter. 

### Дополнительно
Если вы хотите проверить, что всё работает правильно, запустите тесты: введите ``` bash pytest tests/ -v``` и нажмите Enter. Если все тесты прошли успешно, вы увидите надпись "N passed" и зелёные галочки. Если какой-то тест упал, проверьте, что вы находитесь в правильной папке и что все файлы на месте.
